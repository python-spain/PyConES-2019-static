Full Stack Developer - Experto en Python

A3Sec es un grupo multinacional presente en España, México, Colombia y USA, que provee a sus clientes de soluciones en las áreas de Ciberseguridad, Monitorización e Inteligencia de Negocio.

Desde que empezamos en el 2012 no hemos dejado de crecer, incorporando socios y empresas al proyecto para ser más fuertes, más completos y más competitivos en el mercado.

Trabajamos en proyectos punteros dentro de algunas de las más grandes empresas del Mundo siguiendo un modelo de gestión ágil con equipos multidisciplinares repartidos por España, Latinoamérica y USA. Somos innovadores y muy transparentes.

Si te gustan los retos, la tecnología, trabajar con un equipo experto, con metodologías ágiles, en un buen entorno, en oficinas céntricas y bien comunicadas, con horarios flexibles, teletrabajo y que incluso la empresa invierta y apueste en tu formación y talento, A3Sec es tu sitio… Además, tendrás todo el café y Coca-Cola que tu cuerpo necesite.

REQUISITOS

• Experiencia: 2-4 años en backend, frontend y conocimientos de Sistemas • Lenguaje programación: Python • Framework: Django / (Equivalente: Flask) • Control de versiones con Git • Admin. básica Linux, nivel usuario (principalmente Debian y Ubuntu) • Desarrollo y uso de APIs REST • HTML5 + CSS3 • Javascript (ECMAScript 6)

Valoramos conocimiento en las siguientes tecnologías:

• Framework javascript como React.js / (Equivalentes: Angular, Vue.js) • Building Tools (Webpack, NPM) • jquery (para código legacy) • PostCSS o algún preprocesador CSS (SASS, LESS)

Y en las siguientes capacidades relacionadas con la administración de Sistemas:

• Scripting Linux (bash, python, perl, ruby) • Redes IP • Motores de bases de datos: MySQL, MongoDB, Redis, Casandra • Despliegue de configuraciones (salt, puppet, ansible, chef) • Servidores web (apache, nginx, gunicorn, etc.) • Contenedores • Stack ELK (Elastic) • Empaquetado de software (deb y/o rpm)

Ofrecemos:

• Alta remuneración según experiencia, conocimientos y valía • Beneficios Sociales, Horario flexible y Teletrabajo • Indefinido. Jornada completa.
